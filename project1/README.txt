The following goes over how to use phases 1 - 4 of project 1 in CS3502.
Each of these phases can be used in terminal
To compile the C files, use gcc -Wall -pthread phase<number>.c -o phase<number>
to execute you can use ./phase<number>
For phase 2 there is an option to switch from a safe transfer to an unsafe transfer in the teller_thread
All phases should automatically run
